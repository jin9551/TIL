## 🛡️ AWS WAF (Web Application Firewall) 핵심 요약

**AWS WAF**는 AWS 클라우드에서 실행되는 웹 애플리케이션을 **일반적인 웹 취약점 및 익스플로잇**으로부터 보호하는 방화벽 서비스입니다.

> exploit : 악용하다. 부당하게 사용하다.

### 1. 주요 기능 및 보호 대상

* **웹 취약점 방어:** 웹 애플리케이션의 **보안이나 가용성을 저해**할 수 있는 일반적인 웹 익스플로잇으로부터 보호합니다.
* **리소스 과소비 방지:** 애플리케이션이 **과도한 리소스를 소비**하도록 강요하는 공격을 방어하여 불필요한 비용이 발생하는 것을 막아줍니다.
* **향상된 가시성:** 웹 트래픽에 대한 가시성을 높여줍니다.
* **보안 강화:** 웹 공격에 대한 향상된 보안 및 보호 기능을 제공합니다.

### 2. 특징 및 장점

* **배포 용이성:**
    * 배포 및 유지 관리가 쉽습니다.
    * **Amazon CloudFront**를 통해 CDN 솔루션의 일부로 배포하거나, **Amazon API Gateway**를 통해 배포할 수 있습니다.
    * 추가적인 소프트웨어를 배포할 필요가 없습니다.
* **중앙 집중식 관리:** 중앙에서 정의된 규칙을 모든 웹 애플리케이션에서 재사용할 수 있습니다.
* **비용 효율성:** 선불 약정 없이 **사용한 만큼만 지불**하는(Pay-as-you-go) 방식입니다.
* **유연성:** 관리가 용이하고 배포하기 쉬운 **적응력 있는(adaptable)** 웹 애플리케이션 보호 솔루션입니다.

---

AWS WAF는 웹 트래픽을 필터링하여 웹 애플리케이션을 보호하는 **저렴하고 유연하며 관리하기 쉬운** 서비스라고 할 수 있습니다.
